# Projekt Kružnice

## Popis
Projekt obsahuje funkce pro práci s kružnicemi:
- výpočet součtu poloměrů (`radius_sum`)
- výpočet Euklidovské vzdálenosti středů (`euclid_distance`)
- zjištění, zda se kružnice protínají (`has_intersection`)
- vykreslení dvou kružnic (`draw_data` v modulu circles_intersection_draw)
